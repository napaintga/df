◆ dataBuffer

uint8_t* cy_stc_bootload_params_t::dataBuffer

The pointer to a buffer that keeps data to read or write to an NVM.

It is required to be 4-byte aligned.

◆ packetBuffer

uint8_t* cy_stc_bootload_params_t::packetBuffer

The pointer to a buffer that keeps packets sent and received with the Transport API.

It is required to be 4-byte aligned.

◆ initCtl

uint32_t cy_stc_bootload_params_t::initCtl

The initial value to the ctl parameter for Cy_Bootload_ReadData and Cy_Bootload_WriteData.

The Bootloader SDK functions call Read/Write Data functions like this:
Cy_Bootload_ReadData(addr, length, CY_BOOTLOAD_IOCTL_COMPARE, params).
