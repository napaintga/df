◆ errorStatus1

uint32_t cy_stc_crypto_hw_error_t::errorStatus1

[31] - Specifies if ERROR_STATUS0 and ERROR_STATUS1 capture valid error-information.

[26..24] - The error source: "0": INSTR_OPC_ERROR - an instruction decoder error. "1": INSTR_CC_ERROR - an instruction condition code-error. "2": BUS_ERROR - a bus master interface AHB-Lite bus-error. "3": TR_AP_DETECT_ERROR. [23..0] - Captures error description information. For BUS_ERROR:

  • violating the transfer, read the attribute (DATA23[0]).
  • violating the transfer, the size attribute (DATA23[5:4]). "0": an 8-bit transfer; "1": 16 bits transfer; "2": 32-bit transfer.
