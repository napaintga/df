◆ skipMode

uint32_t cy_stc_syspm_callback_t::skipMode

The mask of modes to be skipped during callback execution, see The Defines to skip the callbacks modes.

The corresponding callback mode won't execute if the appropriate define is set. These values can be ORed. If all modes are required to be executed this element should be equal to zero.
