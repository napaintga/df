cy_stc_crypto_server_context_t Struct Reference
Cryptography (Crypto) » Crypto Server » Data Structures

Description

Firmware allocates memory and provides a pointer to this structure in function calls.

Firmware does not write or read values in this structure. The driver uses this structure to store and manipulate the server context.
