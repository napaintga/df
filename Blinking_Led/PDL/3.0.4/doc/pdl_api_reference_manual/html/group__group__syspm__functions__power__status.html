◆ Cy_SysPm_IsLowPower()

__STATIC_INLINE bool Cy_SysPm_IsLowPower ( void  )

Checks whether the device is in Low-Power mode.

Returns
true - the system is in Low-Power mode; false - the system is is not in Low-Power mode.
Function Usage
/* Scenario: there is a need to check if the device is in Low Power mode */
if (Cy_SysPm_IsLowPower())
{
/* The device is in Low Power mode.
* For multi-core devices the Low Power Mode is not core-specific.
* If this function returns "true" - all the cores will indicate
* the Low Power state.
*/
}
