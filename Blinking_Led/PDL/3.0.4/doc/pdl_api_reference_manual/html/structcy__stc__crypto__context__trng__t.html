Cryptography (Crypto) » Crypto Client » Data Structures

Description

Firmware allocates memory and provides a pointer to this structure in function calls.

Firmware does not write or read values in this structure. The driver uses this structure to store and manipulate the TRNG operational context.

◆ GAROPol

uint32_t cy_stc_crypto_context_trng_t::GAROPol

The polynomial for the programmable Galois ring oscillator (TR_GARO_CTL).

The polynomial is represented WITHOUT the high order bit (this bit is always assumed '1'). The polynomial should be aligned so that more significant bits (bit 30 and down) contain the polynomial and less significant bits (bit 0 and up) contain padding '0's.
