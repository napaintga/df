Cryptography (Crypto) » Configuration

General Description

These constants defines an interrupts settings for IPC channel used for the client-server communications.

On the CM4 core crypto driver uses IPC hardware depended interrupt sources.

For CM0+ core user must define the multiplexed interrupt sources by yourself.
