◆ Cy_Crypto_Server_Start()

cy_en_crypto_status_t Cy_Crypto_Server_Start ( cy_stc_crypto_config_t const *  config,
cy_stc_crypto_server_context_t *  context 
)

This function starts the Crypto server on the CM0+ core, sets up an interrupt for the IPC Crypto channel on the CM0+ core, sets up an interrupt to catch Crypto HW errors.

Should be invoked only on CM0.

This function available for CM0+ core only.

Parameters
configThe Crypto configuration structure.
contextThe pointer to the cy_stc_crypto_server_context_t structure that stores the Crypto server context.
Returns
A Crypto status cy_en_crypto_status_t.
